# Pokemon 2.0 Watchface

![](./preview.bmp)

A better skin for Pokemon!

* Wild Pokemon
  * Changes every hour
  * Bar is decreasing in an hour
  * Level is 100 steps
* Owned Pokemon
  * Changes every day
  * Bar is decreasing in a day
  * Level is battery level
  * HP is current date
* Cursor changes every minute

Sprites from: https://veekun.com/dex/downloads
